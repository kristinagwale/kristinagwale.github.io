Challenges/Progress


 At first, we wanted to create a geomap of the US and focus on Drake's stats in each state, but one of the challenges we faced with this idea was the low amount and hard-to-find data for each state individually that we could find. We could only find data for each country as a whole which ultimately led us to switch to a geomap of the world, and show Drake's songs stats in that manner which was good progress and a change of idea. One of the other challenges we faced that prevented us from reaching our full potential was figuring out how to separate the countries (borders to be shown) on the geomap. Instead of only the separated continents worldwide, we wanted all individual countries to be separated as well. With the help of resources, we were able to find out how to include countries' borders. Our progress at this point was stable. We were able to create a final prototype of our project, and start creating a github repository that currently shows a geomap of the world with each country divided, and we are using the prototype as a reference to try to achieve our goal of hovering over countries instead of circles.
